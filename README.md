# مشاريع Arduino التعليمية

هذا المشروع يحتوي على مجموعة من التجارب والمحاكيات البسيطة باستخدام لوحة الأردوينو، وهي مصممة لتكون مرجعاً تعليمياً للمبتدئين في عالم الإلكترونيات والبرمجة.

## قائمة المشاريع

| اسم المشروع | الوصف | رابط Wokwi |
|----------|-------|------------|
| **إنذار صوتي بسيط** | استخدام الطنان (Buzzer) لإصدار أصوات وتنبيهات. | [تجربة المشروع](https://wokwi.com/projects/462402822039286785) |
| **البوابة الذكية** | بوابة تفتح عند الضغط على زر، مع مؤشر ضوئي. | [تجربة المشروع](https://wokwi.com/projects/462400519693322241) |
| **نظام إنذار حساس للحركة** | استخدام حساس PIR لاكتشاف الحركة وتشغيل إنذار. | [تجربة المشروع](https://wokwi.com/projects/462764333130023937) |
| **قفل باب إلكتروني بكلمة مرور** | فتح القفل باستخدام لوحة مفاتيح (Keypad) ورمز سري. | [تجربة المشروع](https://wokwi.com/projects/462868546660159489) |
| **بوابة ذكية بنظام بطاقات RFID** | نظام دخول يعتمد على قراءة بطاقات RFID المصرح بها. | [تجربة المشروع](https://wokwi.com/projects/463198201598476289) |
| **نظام البوابة الذكية بدون لمس** | بوابة تعمل بالموجات فوق الصوتية دون الحاجة للمس. | [تجربة المشروع](https://wokwi.com/projects/462402301401328641) |
| **إشارة المرور** | محاكاة مبسطة لإشارة مرور ضوئية (أحمر/أصفر/أخضر). | [تجربة المشروع](https://wokwi.com/projects/462384760253805569) |
| **التحكم بمصباح LED باستخدام زر ضاغط** | تجربة أساسية للتحكم في LED باستخدام زر (Push Button). | [تجربة المشروع](https://wokwi.com/projects/462382695306434561) |

## كيفية الاستخدام
1. اختر المشروع الذي ترغب في تجربته من الجدول أعلاه.
2. اضغط على رابط Wokwi المرتبط به.
3. سيتم فتح المحاكاة في متصفحك، حيث يمكنك رؤية المكونات والتفاعل معها ومشاهدة الكود البرمجي.
